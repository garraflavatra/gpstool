# gpstool

A web-based tool for retrieving and converting GPS coordinates.

![Screenshot](./.github/screenshot.png)

## Local development

gpstool is developed using [Vite](https://vitejs.dev/) and [Svelte](https://svelte.dev/).

```bash
npm install
npm run dev
```
